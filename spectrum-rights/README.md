# זכות על הרצף — Spectrum Rights Israel

אפליקציית ווב (mobile-first, עברית RTL) לניווט זכויות, מסמכים ובירוקרטיה למשפחות
ואנשים על הרצף האוטיסטי בישראל.

> **חשוב:** האפליקציה אינה מאבחנת אוטיזם ואינה מהווה ייעוץ רפואי או משפטי.
> היא מספקת הכוונה בירוקרטית בלבד, על בסיס מקורות רשמיים (ביטוח לאומי, משרד
> הבריאות, כל זכות ועוד). הזכאות הסופית נקבעת תמיד על ידי הגורם הרשמי.

## מה יש ב-MVP

- **שאלון חכם** — שאלה אחת בכל מסך: גיל, מצב אבחון, קצבה, עיר, קופה, מסגרת,
  חשבונות בית, מצב תפקודי, מסמכים קיימים.
- **מנוע זכאות** — מחשב אילו זכויות רלוונטיות: גמלת ילד נכה (+תזכורת
  רטרואקטיבית), טיפולים פרא-רפואיים, טב״ם, הנחת מים, הנחת ארנונה, נקודות זיכוי
  במס, תג חניה, ועדת זכאות ואפיון, נכות כללית ושירותים מיוחדים לבוגרים.
- **"מה מגיע לי עכשיו?"** — דשבורד עם "המשימה הבאה" אחת, לא רשימה מציפה.
- **מצב "אני מוצף"** — מסך שמראה רק את הצעד הבא, ותו לא.
- **תיק מסמכים** — העלאת PDF/תמונות, סיווג לפי סוג, זיהוי מסמכים חסרים.
- **מחולל מכתבים** — הפניה לקופה, השלמת מסמכים לביטוח לאומי, בקשת ארנונה,
  בקשת מים, נקודות זיכוי, פנייה לוועדת זכאות, וערעור.
- **מעקב סטטוס** — "הגשתי — מה עכשיו?": לא התחלתי / בתהליך / הגשתי / ביקשו
  מסמכים / אושר / נדחה — עם הצעד הבא המתאים לכל מצב.
- **מקורות** — לכל זכות: שם מקור, קישור רשמי, תאריך בדיקה אחרון ורמת ביטחון.

## פרטיות

כל הנתונים — הפרופיל, הסטטוסים והמסמכים — נשמרים **במכשיר בלבד**
(localStorage + IndexedDB). שום דבר לא נשלח לשרת. מתאים במיוחד למסמכים
רפואיים רגישים, ומאפשר לפרוס את האפליקציה כאתר סטטי בלבד.

## הרצה

```bash
cd spectrum-rights
npm install
npm run dev      # פיתוח
npm run build    # בנייה ל-dist/
```

## סטאק

React 18 + TypeScript + Vite + Tailwind CSS 4 + React Router (HashRouter —
עובד מכל אחסון סטטי ללא קונפיגורציית שרת).

## מה הלאה (מחוץ ל-MVP)

- OCR וסיווג אוטומטי של מסמכים
- לוח דד-ליינים ותזכורות (דורש backend / התראות)
- "פענוח מכתב" — העלאת מכתב תשובה וקבלת הסבר פשוט + פעולה הבאה
- סנכרון בין מכשירים וחשבון משפחתי (דורש backend + הצפנה)
- מאגר כללי זכאות מנוהל עם תהליך אימות מקורות תקופתי
