# WhatsPlan

אפליקציית Android פרטית שמחלצת חזרות, פגישות, הופעות והקלטות
מייצוא שיחות WhatsApp ומהתראות חדשות. כל אירוע מוצג לבדיקה לפני פתיחת
מסך היצירה ביומן.

## פרטיות

- אין גישה למסד הנתונים של WhatsApp.
- אין Root, הנדסה לאחור או עקיפת הצפנה.
- תוכן השיחות נשמר מקומית בלבד ב-SQLite.
- האפליקציה אינה כותבת ישירות ליומן; היא פותחת טופס אירוע מוכן.
- זיהוי קבוצה משתמש ב-MessagingStyle, כותרת השיחה ומזהה Shortcut קבוע.
- מזהה הקבוצה הוא Hash מקומי; שמות והודעות אינם חלק מהמזהה הנשמר.

## שימוש

1. ב-WhatsApp פותחים שיחה.
2. שלוש נקודות > עוד > ייצוא צ'אט > ללא מדיה.
3. משתפים ל-WhatsPlan.
4. בודקים את האירועים ולוחצים "פתח ביומן".
5. לקליטת הודעות עתידיות לוחצים "התראות חדשות" ומאשרים גישה.

## בנייה

נדרש Android Studio עם Android SDK 35 ו-JDK 17.

```bash
./gradlew assembleDebug
```

ה-APK יופיע תחת:

`app/build/outputs/apk/debug/app-debug.apk`
